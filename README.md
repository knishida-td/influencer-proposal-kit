# influencer-proposal-kit

インフルエンサー向け商品企画提案資料の生成ツール。

## 構成パターン

インフルエンサーへの商品コラボ提案に特化したスライド構成:

1. **表紙** — 商品ビジュアル + タイトル
2. **コンセプト** — なぜこの商品をこの人に提案するか
3. **商品概要** — 商品写真 + セールスポイント
4. **機能性** — スペック詳細（素材・技術）
5. **品質証明** — 認証・エビデンス
6. **競合比較** — 他ブランドとのスペック比較表
7. **生産背景** — 工場実績・製造クオリティ
8. **デザインイメージ** — 商品デザインの詳細
9. **カラー・サイズ展開** — バリエーション
10. **販売戦略** — プロモーション計画
11. **スケジュール** — 製造〜リリースのタイムライン
12. **販売イメージ** — 価格・数量・売上サマリー
